# NEXTGEN AUTOMATION OFFICE

First Run the following command to install the required packages:

```bash
pip install -r requirements.txt
```

Then run the following command to start the server:

```bash
python main.py
```

Edit UI please install Qt Designer and edit the .ui files in the UI folder.

Run file convertUitoPy to convert the .ui files to .py files (Apply update UI). And edit line 26 of file MainWindow_ui.py to 'import src.ui.icon_rc'
